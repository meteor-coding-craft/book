# 2장 개발 환경 설정

미티어는 Node.js 기반의 풀스택 플랫폼이지만 별도로 Node.js의 설치를 신경 쓸 필요가 없다. 몽고DB는 프로젝트를 생성하면 자동으로 내장되어 실행까지 된다. 현재 미티어는 유닉스, 리눅스 환경에서의 설치 환경을 지원하며 윈도 환경에서는 설치 파일을 내려받기하여 사용할 수 있다.
