# 개요

각장의 타이틀명에 괄호 표시가 있습니다. 그 의미는 다음과 같습니다. 

| 꼬리표 | 의미 |
| :---: | :--- |
| \(old\) | 책과 내용이 같음 - 즉 1.5 버전 입니다. |
| \(empty\) | 아직 내용을 채우지 않았음 |
| \(ing\) | 현재 작업중 |
| \(new\) 또는 없음 | 1.10 이상 버전으로 업데이트 완료됨 |
